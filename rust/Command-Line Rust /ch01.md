# ch01

## cargo command

```bash
# create
linux:~ $ cargo new hello
linux:~ $ tree hello/
hello/
├── Cargo.toml
└── src
    └── main.rs
linux:~/hello $ cat src/main.rs
fn main() {
    println!("Hello, world!");
}

# build
linux:~/hello $ cargo build
linux:~/hello $ ls target

# run
linux:~/hello $ cargo run

# test
linux:~/hello $ cat << EOF > tests/test.rs
#[test]
fn work() {
    assert!(false);
}
EOF
linux:~/hello $ cargo test

# depend
linux:~/hello $ cat Cargo.toml
[package]
name = "hello"
version = "0.1.0"
edition = "2021"

[dependencies]

linux:~/hello $ vi Cargo.toml
[dependencies]
assert_cmd = 1

linux:~/hello $ cargo test
```
